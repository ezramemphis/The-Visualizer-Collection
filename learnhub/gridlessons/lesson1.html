Learning How This Visualizer Works

Step 1: Just the Canvas

First, we grab the <canvas> element and its 2D drawing context (ctx). Think of ctx as our “pen” that can draw on the canvas. 


const canvas = document.getElementById('canvas');
const ctx = canvas.getContext('2d');

Step 2: Loop to Create a Grid

To make the dots appear, we repeat a small drawing command many times using two loops. The first loop moves across the canvas from left to right (the x direction), and the second loop moves from top to bottom (the y direction). Together these loops visit every position in a grid.

For each position, we calculate where the dot should go using x * stepX and y * stepY. These values spread the dots evenly across the canvas. Finally, we use ctx.fillRect() to draw a tiny 4×4 square at that location. Because this happens many times in the loops, lots of tiny squares appear, which look like a grid of dots. 


for(let x = 0; x < grid; x++){
  for(let y = 0; y < grid; y++){

    // calculate the position
    const posX = x * stepX
    const posY = y * stepY

    // draw a tiny square (our "dot")
    ctx.fillRect(posX, posY, 4, 4)

  }
}

Step 3: Watch the Code Change

Move the slider and watch the grid variable update in the code panel. This is the exact number controlling how many squares appear.

Step 4: Changing the Shape of the Dots

Right now every point in the grid could be drawn the same way, but we made the visualizer more interactive by allowing the user to choose what shape each dot should be. This is done using the dropdown menu. When the dropdown changes, JavaScript listens for that event and updates a variable called shape.

Instead of drawing squares directly inside the loop, the program calls a helper function called drawShape(). This function receives the position of the dot and decides what shape to draw depending on the value stored in shape. A switch statement is used so the program can easily choose between squares, circles, triangles, and other shapes. 


shapeSelect.addEventListener("change", () => {

  shape = shapeSelect.value
  draw()

})


function drawShape(x,y,size){

  switch(shape){

    case "square":
      ctx.fillRect(x,y,size,size)
    break

    case "circle":
      ctx.beginPath()
      ctx.arc(x+size/2,y+size/2,size/2,0,Math.PI*2)
      ctx.fill()
    break

  }

}

Step 5: Drawing the Connecting Lines

The connecting lines are controlled by a checkbox. When the checkbox is turned on, the program draws lines between neighboring points in the grid. We first check whether the checkbox is active using checkbox.checked. If it is true, we run the code that draws the lines.

To connect the grid points, the program uses two sets of loops. The first set draws horizontal lines between dots from left to right. The second set draws vertical lines connecting dots from top to bottom. Together these loops create a grid-like network that reveals the structure of the points.
