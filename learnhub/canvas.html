Lesson 1: Setting Up the Canvas

Getting Started with the Canvas


Step 1: Add the Canvas to HTML

The <canvas> element is where all our drawing will happen. Here we give it a width and height so it has a visible space on the page. Think of it as an empty blackboard where your visualizer will be drawn. 

<canvas id="canvas" width="600" height="400"></canvas>

Step 2: Include CSS for Canvas

We style the canvas to look nice and visually stand out. A border, shadow, and background color make it clear where the drawing will happen. Although technically not necessary, especially if you intend the canvas to fill the entire screen, but it's good to know. 

canvas {
  display: block;
  background: #000;
  margin: 20px auto;
  border: 2px solid #0ff;
  box-shadow: 0 0 30px #0ff88aa;
  border-radius: 12px;
}

Step 3: Grab the Canvas in JavaScript

To start drawing, we need the canvas in our JavaScript. We grab it using document.getElementById and then get its 2D context with getContext('2d'). This context is like a pen for the canvas. 

const canvas = document.getElementById('canvas');
const ctx = canvas.getContext('2d');

Step 4: Set Up for Future Drawing

Even though we won’t draw anything yet, it’s good to define variables for future use, like the grid size or animation settings. This helps beginners understand how to structure code for interactivity later. 

let grid = 10;  // Number of cells (we'll use this in Lesson 2)
let showLines = false;  // Optional toggle for connecting lines

Step 5: Structure Your HTML for UI Controls

Even though we’re not making interactive sliders yet, it’s helpful to leave space in the HTML for them. This teaches beginners good page structure for future visualizers. 

<div class="ui">
  <!-- Slider or buttons will go here in future lessons -->
</div>
